# Mi Proyecto Git

## Objetivos:
Este proyecto forma parte de la práctica **"Explorando Git y GitHub paso a paso"**, cuyo objetivo es aprender a utilizar Git y GitHub de forma completa: desde la creación de repositorios hasta el manejo de ramas, fusiones (*merge*) y rebases (*rebase*).  

## Incluye:
Incluye ejemplos prácticos realizados con **GitHub Desktop en Windows**, sin usar la consola, para facilitar el aprendizaje visual y la gestión de versiones.
